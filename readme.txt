A WordPress plugin to display your customers comments. 
  
== Description ==
  
Basic comments/testimonials plugin that takes the coments from your Hotel and puts in your web page adapting the style to the theme.
Also allows to set a specific number of comments that can be called with another shortcode.
  
== Installation ==
  
Upload the plugin files into the /wp-content/plugins/alojapro-widget directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the ‘Plugins’ screen in WordPress
Available features:

To get all the comments:
1. Use the Alojapro Comments option on the sidebar menu to configure the Property Id.
2. Add the shortcode [getComments] on the content of a page/post/etc. to add the plugin into a page.

To use a specific number of comments:
1. Use the Iframe settings option on the sidebar menu (Comments) to configure the iframe parameters.
2. Add the shortcode [getNumberOfComments] on the content of a page/post/etc. to add the desired number of comments
